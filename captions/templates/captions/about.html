About This Project

This Image Caption Generator is built using a VisionEncoderDecoder (ViT + GPT-2) model to automatically describe images in natural language. It integrates:

  • Computer Vision for feature extraction
  • Natural Language Generation for sentence formation
  • Multilingual Translation using transformers
  • Text-to-Speech synthesis for audio captions

This project demonstrates advanced understanding of Deep Learning pipelines, model deployment, and AI-driven user interfaces using Django.
